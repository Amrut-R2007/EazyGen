# EazyGen ⚡

**EazyGen** is an ultra-fast, premium, single-page browser utility hub designed to handle universal file conversions, comprehensive PDF manipulation, custom-sized file compression, and instant QR code generation. 

Engineered with a strict serverless, client-side architecture by **ByteCraft Studio**, EazyGen processes everything locally in the browser—ensuring your data never touches a third-party server.

---

## 🔒 Privacy & Performance Architecture

*   **100% Serverless & Local:** All file rendering, converting, and compressing happens directly inside the user's browser canvas and memory space. 
*   **Zero Database Footprint:** We never upload, view, store, or track your documents. 
*   **Full Offline Capability (PWA):** Equipped with a robust Service Worker caching architecture, EazyGen runs completely offline. Once loaded, you can run the entire suite mid-flight or without an active internet connection.

---

## 🛠️ Feature Catalog

### 1. Advanced PDF Suite (From Reference Specifications)
*   **Organize:** Merge, Split, Rotate, Crop, and dynamically re-order PDF pages.
*   **Security:** Protect PDFs with custom passwords, Unlock secured documents, and permanently Redact sensitive text or graphics.
*   **Enhancements:** Stamp customizable text/image Watermarks and inject automated Page Numbers.
*   **Optimization & Repair:** Compress PDF files locally or run localized document health checks to repair corrupted data structures.

### 2. Universal Conversion Toolkit
*   **Two-Way Office Formats:** Seamless conversion pipelines between PDF and Word, PowerPoint, Excel, or high-res JPGs.
*   **Advanced Formats:** Convert standard files directly into clean Markdown text layouts (optimized for LLM ingestion) or transform documents to the archiving-standard PDF/A format.
*   **Media Extensions:** A fully local drag-and-drop Image Converter (PNG, JPG, WEBP, GIF, HEIC) and a dedicated Audio Extractor to strip raw MP3 files from MP4 video uploads.

### 3. AI & Advanced Modules (Client-Side Simulations)
*   **AI Summarizer & Translator:** Generate immediate structural bullet points or translate document text while preserving native layouts.
*   **OCR Engine:** Convert flat, scanned PDF assets into completely searchable, editable text layouts.
*   **Compare & Scan:** Visual side-by-side comparison interfaces for checking modifications across document versions.

### 4. Custom Local Utilities
*   **Custom File Compressor:** Includes a variable slider enabling users to target strict maximum output sizes (MB/KB) to meet custom compression criteria perfectly.
*   **QR Code Generator:** Input any URL target to instantly build, render, and extract high-resolution PNG QR codes directly from a clean local canvas element.

## 💻 Technical Setup & Stack

*   **Frontend UI:** Vanilla JavaScript / modern React, Tailwind CSS framework.
*   **Icons:** Crisp, modern vectors via Lucide Icons.
*   **Offline Engine:** Custom vanilla Service Worker script using a Cache-First asset delivery strategy.

### Local Development
To launch EazyGen locally:
1. Clone the repository: `git clone https://github.com/your-username/eazygen.git`
2. Open `index.html` in any modern web browser, or spin up a simple static local server.
